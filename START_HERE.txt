FIRST TIME SETUP (ONE-TIME):
----------------------------
1. Double-click:  setup.bat
   (Installs dependencies and creates environment)

2. Wait for "Setup completed successfully!" message


LAUNCH THE BOT:
---------------
1. Double-click:  start.bat

2. Open browser:  http://localhost:8000

3. Add your CSFloat account in web interface

4. Create buy orders on CSFloat website

5. Click "Start" button in web interface


QUICK LAUNCH (after setup):
----------------------------
Double-click:  quick-start.bat


BATCH FILES OVERVIEW:
---------------------
setup.bat       - Initial setup (run once)
start.bat       - Normal launch with checks
quick-start.bat - Fast launch without checks


WEB INTERFACE STEPS:
--------------------
1. Add Account:
   - Name: Your account name
   - API Key: From CSFloat website
   - Proxy: Optional (http://user:pass@ip:port)

2. Settings:
   - Check Interval: 120-180 seconds
   - Outbid Step: $0.01-$0.05
   - Max Outbids: 5-10

3. Create Buy Orders:
   - Go to CSFloat website
   - Create buy orders manually
   - Bot will detect them automatically

4. Start Bot:
   - Click "Start" button
   - Watch automatic outbidding!

TROUBLESHOOTING:
----------------
Q: Setup fails?
A: Check Python 3.10+ installed: python --version

Q: Bot doesn't start?
A: Check logs in logs/ directory

Q: Don't see orders?
A: Create orders on CSFloat website first

Q: Dependencies error?
A: Run setup.bat again
